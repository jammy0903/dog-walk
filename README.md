# 강아지 산책 — 타자로 걷기 (크롬 확장 v2)

키보드 **한 글자 = 강아지의 한 걸음**. 아무 사이트에서나 타자를 칠수록 강아지가 걷고,
**걸음을 🦴돈으로 환전**해 **새 강아지(품종)를 입양·수집**한다. v1(aingan.click, 감정 모으기)과 **완전 독립**.

## 게임 루프

타이핑 → 걸음(steps) → `환전` 버튼(1걸음=1🦴) → 코인 → 🐕 강아지 상점에서 품종 입양 → 선택하면 통째 교체.
(11품종: 치즈/크림/참깨 시바·웰시코기·치와와·토이푸들·프렌치불독·보더콜리·차우차우·이탈리안그레이하운드·골든리트리버.)
페이지에 풀어놓으면 펫이 **화면 테두리(바닥·양벽·천장)를 한 바퀴** 돌며 산책.
※ 꾸미기(액세서리 착용)는 옆모습 위치 맞추기가 불안정해 폐기 → 강아지 수집으로 전환.

## 핵심 원칙

- **개인정보**: keydown '개수'만 센다. 무슨 키인지(`e.key`)는 절대 안 본다 = 키로거 아님.
- **공유**: 백그라운드 + `chrome.storage` 하나가 진실의 원천 → 모든 탭/창이 같은 게임 공유.
- **idle 없음**: 자동 걷기 없다. **키보드 칠 때만** 걷는다 — 친 만큼만 정확히.
- **저장(계정연동)**: `chrome.storage.local`(즉시) + `chrome.storage.sync`(구글 계정에 묶여 기기끼리
  동기화·15초 throttle). 시작 시 `taps`(단조 증가) 큰 쪽으로 병합. (풀 OAuth는 추후.)
  - **세이브 견고성(v0.3.2)**: 읽기·병합 시 `normalize()`로 타입/범위 강제 + 미지 필드 제거 →
    손상·오염된 sync 레코드로부터 NaN 전염·배율 폭주·병합 동결을 차단.
    ⚠️ 서버리스라 *소유자 본인 치트*는 구조적으로 차단 불가(이건 견고성 목적이지 안티치트가 아님).
- **아이콘 = 팝업(v0.3.0)**: 툴바 아이콘 클릭 → 작은 팝업. ① **강아지 표시 ON/OFF 토글**
  (끄면 강아지만 숨고 **걸음수는 그대로 적립**, 아이콘에 `OFF` 뱃지) ② **게임 열기**(사이드패널).
- **동영상 중엔 자동 숨김(v0.3.0)**: 전체화면이거나 큰 동영상(임베드 iframe 포함)이 재생 중이면
  강아지를 가려 시청을 방해하지 않는다(멈추면 그 자리에서 재등장). 임베드는 자식 프레임이
  `chrome.runtime`으로 background에 보고 → TOP 프레임에만 중계(웹페이지 위·변조 불가 채널).

## 크롬에 올리기 (개발자 모드)

1. `chrome://extensions` → 우상단 **개발자 모드** ON
2. **압축해제된 확장 프로그램을 로드** → 이 `extension/` 폴더 선택
3. 툴바 아이콘 클릭 → 팝업(강아지 ON/OFF·게임 열기). 게임(환전·상점)은 사이드패널로 열림
4. 아무 웹페이지에서 타이핑 → "걸음" 오름 → 게임 열기 → 환전 → 상점에서 품종 입양

## 설치 못 할 때(DLP·관리정책 등) — 로컬 미리보기

확장 설치/개발자모드가 막혀도, **일반 로컬 웹페이지로 미리볼 수 있다**(`dev-preview.html`).
`chrome.*` API를 가짜로 채워(shim) 게임 코드를 그대로 돌린다 — DLP 우회 아님, 그냥 로컬 페이지.

```bash
python3 -m http.server 8123 --directory <repo>/extension
# 브라우저에서 http://localhost:8123/dev-preview.html 접속 → 아무 키나 치면 강아지가 걸음
```

- 검증됨: 강아지 걷기(키1=걸음+1, 1:1)·배경·상점·환전·페이지펫 전부 동작.
- 한계: 저장이 그 브라우저 localStorage뿐(기기간 sync 아님), 펫은 미리보기 페이지에서만(다른 사이트 X).
- ⚠️ `dev-preview.html`은 DEV 전용 — 웹스토어 zip엔 넣지 말 것.

## 파일

| 파일 | 역할 |
|---|---|
| `manifest.json` | MV3 (storage·sidePanel·action 팝업·content_script `<all_urls>`) |
| `content.js` | 모든 페이지 keydown/클릭 개수만 카운트(`isTrusted`만) → 전송 + 화면 테두리 펫 + 동영상 중 숨김 |
| `background.js` | 진실의 원천: 걷기·환전·구매 로직 + 상점 + local/sync 저장·`normalize` 병합 + 동영상 보고 중계 |
| `game.html/js` | 강아지·걸음·환전·상점·배경·언어(한/영) 화면(사이드패널) |
| `popup.html/js` | 아이콘 팝업: 강아지 표시 ON/OFF 토글 + 게임 열기 |

## 현재 단계

- **Stage (지금, v0.3.2)**: 실사 누끼 강아지 + **키/클릭 1번=걸음 +1(1:1, `isTrusted`만)** + 걷기 4프레임 핑퐁
  + 환전 + **강아지 상점(11품종)** + 한/영 + 계정 동기화(정규화 병합)

  + **아이콘 팝업 ON/OFF 토글** + **동영상(전체화면·임베드) 중 자동 숨김**.
